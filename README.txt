# Beerotheque

## Description

Ce projet avait un but éducatif. Il permet de lister et de trier sous la forme d'un CRUD n'importe quelle liste d'objets (ici des bières). Il permet de lister, trier, modifier, supprimer et ajouter avec en plus un système de pagination. Du Java Spring Boot a été utilisé pour le backend et du Vue Js pour le frontend avec une version javascript et une autre typescript.
